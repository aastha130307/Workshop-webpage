# 🤖 AI & Robotics Workshop Webpage

A modern, responsive, and interactive website built for promoting an **AI & Robotics Workshop**. The platform provides complete workshop information, curriculum details, instructor profiles, FAQs, testimonials, and a seamless registration experience for participants.

The project is designed to offer an engaging user experience with clean UI components, smooth navigation, and mobile-friendly responsiveness.

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/aastha130307/Workshop-webpage.git
```

### 2. Navigate to the Project Directory

```bash
cd Workshop-webpage
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Start Development Server

```bash
npm run dev
```

The application will run locally on:

```bash
http://localhost:5173
```

## 🏗️ Production Build

Create an optimized production build:
